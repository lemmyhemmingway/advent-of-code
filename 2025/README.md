# Advent of Code 2025

Solutions in multiple programming languages:

- **Rust** - `rust/` - Performance-focused
- **Python** - `python/` - Quick prototyping
- **Go** - `go/` - Concurrent solutions
- **Elixir** - `elixir/` - Functional approach

## Running Solutions

### Rust
```bash
cd 2025/rust
cargo run --release -- <day>
```

### Python
```bash
cd 2025/python
python day01.py
```

### Go
```bash
cd 2025/go
go run main.go <day>
```

### Elixir
```bash
cd 2025/elixir
mix escript.build
./_build/dev/escripts/aoc <day>
```

## Input Files

Input files are stored in `inputs/` directory and named `dayXX.txt`.

Get your input from [adventofcode.com](https://adventofcode.com/2025)
